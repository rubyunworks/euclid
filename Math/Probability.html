theil_index(array) click to toggle source

Calculates the Theil index (a statistic used to measure economic inequality).

TI = sum_{i=1}^N frac{x_i}{sum_{j=1}^N x_j} ln frac{x_i}{bar{x}} 

  http://en.wikipedia.org/wiki/Theil_index
     # File lib/euclid/math/probability.rb, line 187
187:     def theil_index(array)
188:             return 1 if array.size <= 0 or any? { |x| x < 0 }
189:             return  0 if array.size <  2 or all? { |x| Math::float_equal(x, 0) }
190:             m = mean(array)
191:             s = sum(array).to_f
192:             inject(0) do |theil, xi|
193:                     theil + ((xi > 0) ? (Math::log(xi.to_f/m) * xi.to_f/s) : 0.0)
194:             end
195:     end
