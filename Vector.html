Vector

The Vector class represents a mathematical vector, which is useful in its own right, and also constitutes a row or column of a Matrix.

house() click to toggle source

Computes the Householder vector (MC, Golub, p. 210, algorithm 5.1.1) 

     # File lib/euclid/vector.rb, line 158
158:   def house
159:     s = self[1..length-1]
160:     sigma = s.inner_product(s)
161:     v = clone; v[0] = 1
162:     if sigma == 0
163:       beta = 0
164:     else
165:       mu = Math.sqrt(self[0] ** 2 + sigma)
166:       if self[0] <= 0
167:         v[0] = self[0] - mu
168:       else
169:         v[0] = - sigma.quo(self[0] + mu)
170:       end
171:       v2 = v[0] ** 2
172:       beta = 2 * v2.quo(sigma + v2)
173:       v /= v[0]
174:     end
175:     return v, beta
176:   end
